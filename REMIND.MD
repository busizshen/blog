npm install hexo -g，开始安装Hexo

hexo -v，检查hexo是否安装成功

hexo init，初始化

npm install，安装所需要的组件

hexo g，首次体验Hexo

hexo s，开启服务器

hexo new post “博客名” 新建博客

hexo d -g，生成以及部署

需要安装一个扩展：npm install hexo-deployer-git --save
